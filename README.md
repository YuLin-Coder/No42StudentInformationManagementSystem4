## 项目介绍
基于ssm的学生信息管理系统：前端jsp、jquery、easyUI，后端 spring、mybatis，集成学生信息管理、班级管理、年级管理等功能于一体的系统。

## 功能介绍
![](screenshot/0.png)

- 增删查改、分页、模糊查询、权限管理、登录拦截

- 管理员列表（增加、查看、修改、删除、批量删除）
- 年级列表（增加、查看、修改、删除、批量删除）
- 班级列表（增加、查看、修改、删除、批量删除）
- 学生列表（修改）
